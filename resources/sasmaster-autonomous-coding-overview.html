Core principle: The human gate is inviolable. In Phase II, the agent shifts from passive executor to intelligent observer — it reads system state, infers what should be built next, and writes the spec. But no task executes without explicit human approval. Shiv remains the pure decision-maker; the agent eliminates the cognitive load of observation and spec authorship.
