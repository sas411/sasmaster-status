Wikidata 5-Signal Certification Model
Rank (wikibase:rank)
Claim deprecated? Blocks immediately.
QUARANTINE
Entity Cardinality
QID carries >1 distinct IMDB ID?
QUARANTINE
Cross-Entity Cardinality
One IMDB claimed by >1 QID? Merge candidate.
QUARANTINE
Type Agreement (P31)
Entity is film/TV? Guards nm/ch trap (Q5=human).
QUARANTINE
Sourcing / Corroboration
Referenced + ≥2 corroborating systems? (non-blocking)
PROVISIONAL
✓ CERTIFIED
All 5 pass · sourced · ≥2 corroborators
→ Auto spine + eidr backfill
~ PROVISIONAL
Clean but unsourced / sparse
→ Hold · sample audit first
✗ QUARANTINE
Any blocking flag → adjudication queue
→ Human only · never auto-promoted
