Five Signature Experiences
Hero Hiker
Full-viewport cinematic title browser. TMDB poster as hero. Horizontal scroll. Hover reveals metadata overlay. Click expands to Program Drill-Down. Apple TV+ × Netflix editorial × Bloomberg density.
Dr. Scoop — AI Companion
Floating panel (bottom-right). Media insider voice — direct, analytically grounded, occasionally wry. claude-sonnet-4-6 · SSE stream · POST /api/scoop/chat · Prompt caching enabled for strategy context.
Data Viz Suite
Ratings timelines (C3/C7 trend by episode), audience composition wheels (demo radial), rights availability matrix (title × territory × window), content performance scatter (audience × sentiment × cost), DMA heat maps.
Program Drill-Down
All SaSMaster data on one title. Six tabs: Identity (all IDs) → Audience (Nielsen history) → Content (cast, crew, genres) → Rights (territory map) → Intelligence (Dr. Scoop) → Comparables (related titles). Sub-second load via DuckDB.
Metadata Booklet
Printable one-pager for any title. Parent Key IDs, core metadata grid, audience summary, availability status, Dr. Scoop analysis. Output: screen HTML, PDF export, PNG thumbnail. Replaces the manual "compile a fact sheet" workflow.

Competitive Positioning
vs. Gracenote / TMS
Deepest program guide data but no audience intelligence, no rights layer, no AI. SaSMaster wraps Gracenote data with intelligence layers Gracenote can't provide.
vs. Nielsen ONE
Owns the measurement currency but internal-facing UI, enterprise-only pricing. SaSMaster democratizes Nielsen data within a licensed framework.
vs. Parrot Analytics
Demand analytics but no operational data (rights, deals, schedules). Research tool, not workflow tool. SaSMaster is both.
vs. Canvs AI / sentiment tools
Narrow scope — social sentiment only. Dr. Scoop delivers broader intelligence: audience + rights + competitive context + financial performance.

Our wedge: The Parent Key. No competitor has a unified content identifier spanning IMDB + TMDB + EIDR. This bridge puts SaSMaster at the center of a data graph that others can't replicate without years of matching work.
