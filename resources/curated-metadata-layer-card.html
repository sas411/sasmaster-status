Today's live spine is 100% CERTIFIED tier (fusion v1); the CONFIRMED row activates as scoring v2 bands land on the spine. The gate logic is band-driven, so no curated-layer change is needed.

Why the IMDB Exception Is Safe

imdb_tconst is the mint anchor — immutable. An IMDB-only record (score 0.50) can only ever be enriched when TMDB / EIDR corroboration lands later. It is never rekeyed. Admitting it adds long-tail coverage with zero join risk.
TMDB orphans are the opposite. Their synthetic tmdb_mov_ / tmdb_tv_ keys CAN be rekeyed when an IMDB match arrives — which would break every downstream join. They stay HELD until promoted.

Every curated row carries cert_band · anchor_basis · match_class — so product surfaces filter Certified-only (the governance contract), while analytics opts into Confirmed + Provisional for maximum coverage. The choice moves to the consumer; the provenance never leaves the row.

QA Gates & Telemetry

— no self-certification · communication is key · every build leaves a trail

Three Gates · Pass or No Snapshot

G1
Gate integrity — zero curated rows without an imdb / eidr anchor; every row carries a valid cert_band. The HELD wall is verified, not assumed.
G2
Key integrity — imdb_id unique per title master, nconst unique in people; table counts tie out exactly to the eligibility gate. Validated 06-11: 0 duplicates across 1.27M title rows.
G3
Idempotency + honest telemetry — re-run without a new cert run is a no-op; every build logged to curated.build_log; Slack-posted counts must equal actual table counts.
⟳ Fail-closed by design: any gate failure marks the build FAILED, withholds the S3 snapshot, keeps the previous good tables serving, and pages #builds. A broken build can never silently replace a good one.

06

Governance & Guarantees

— same contract as the spine · extended to the consumable layer
✓ Band on Every Row
cert_band · anchor_basis · match_class travel with every record. Product filters Certified-only; analytics chooses coverage. Provenance never leaves the row.
✗ Held Never Enters
TMDB orphans and quarantine are excluded by construction — rekeyable synthetic keys cannot pollute business joins. Promotion is the only door in.
⛁ Derived, Never Patched
Full rebuild from the certified spine each cert run. SCD2 history stays on the spine; curated is always its freshest projection. No drift possible.
⚖ Fail-Closed QA
Three gates run as SQL assertions inside the build. Failure = no snapshot, previous tables keep serving, immediate page. The agent never self-certifies.
