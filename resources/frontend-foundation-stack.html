→packages/tokens owns all CSS vars — apps only import
→packages/ui owns Tailwind v4 + PostCSS — one config, zero duplication
→All components import from @sasmaster/ui — never inline Tailwind that duplicates a token
→Storybook story required for every new component in packages/ui
→App Router only — no Pages Router files, ever
→Highcharts components always 'use client'
