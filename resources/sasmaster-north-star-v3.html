Before Building Anything New
THE SASMASTER TEST — 5 QUESTIONS
P1 MEMORY
Does this persist context that would otherwise be re-learned?
P2 LEARNING
Does this create reusable capability, not just one-shot output?
P3 TRANSPARENCY
Can Shiv inspect what it's doing without asking someone?
P4 PURPOSE
Is there a named consumer for this output?
P5 GROWTH
Will the system be more capable after this ships than before?
