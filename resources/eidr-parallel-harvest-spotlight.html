THE CEILING WE HIT

// The Constraint

One workstation can't fan out

EIDR enrichment needs two layers — bulk L1 metadata and browser-extracted L3 Alternate IDs. The browser lane is the bottleneck.

  • macOS throttles concurrent launches of the same browser binary — only 1 of 3 Camoufox bots would reliably start.
  • Stale profile locks & geoip contention stalled every multi-bot attempt.
  • Single-bot ceiling: ~225 rec/hr — leaving 306K titles effectively unreachable at scale.
// The Move

Lift the bots to the cloud

Push the browser fleet onto a Linux EC2 instance where each bot owns an isolated virtual display — no OS throttle, true parallelism.

  • 3 bots on separate Xvfb displays :99 / :100 / :101 — no contention.
  • L1 metadata sweep runs alongside at ~31,700 rec/hr across 5 threads.
  • Every record is checkpointed — a crash loses zero work, the run resumes itself.
